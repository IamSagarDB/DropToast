## Dependency

Add it in your root build.gradle at the end of repositories:

```
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
		}
	}
  
  ```

Add dependency in your app module

```
dependencies {
	 implementation 'com.github.sagargowda7:DropToast:0.0.1'
	}
}
```

## Usage

### Java
```
 DropToast.makeText(MainActivity.this," Your message ",DropToast.LONG,DropToast.SUCCESS);
```
### Features

#### Duration 

For Long duration.
```
DropToast.LONG 
```
For short duration.
```
DropToast.SHORT
```

#### Type 

for SUCCESS message
```
DropToast.SUCCESS 
```
for ERROR message
```
DropToast.ERROR
```
for WARNING message
```
DropToast.WARNING
```
